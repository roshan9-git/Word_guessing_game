# Password Guessing Game

A simple Python console game where the player has to guess a secret word.  
The game provides hints by showing correctly guessed letters in the right positions.

## Features
- Choose difficulty level: easy, medium, or difficult
- Hints show correct letters in their positions
- Counts the number of attempts
- Fun and educational for practicing vocabulary

## How to Play
1. Run the Python program (`python game.py` or the filename you used)
2. Choose a difficulty level: `easy`, `medium`, or `difficult`
3. Try to guess the secret word
4. Hints will show which letters you got correct
5. Keep guessing until you find the secret word

## Word Lists
- **Easy**: simple, common words like `happy`, `run`, `play`, etc.
- **Medium**: slightly challenging words like `analyze`, `challenge`, `observe`, etc.
- **Medium-Difficult**: tricky words like `ambitious`, `strategic`, `adaptable`, etc.

## Requirements
- Python 3.x installed

## Example
Enter difficulty level (easy/medium/difficult): medium
Guess the secret password! (It has 7 letters)
Enter your guess: analyze
Hint: a____e
......
Congratulations! You guessed it in 4 attempts!
